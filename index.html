Figma

Inicialmente, antes de aprender até mesmo linguagens de marcação, conhecemos o Figma, uma ferramenta de design colaborativa que permite que os usuários possam desenvolver a interface de seus sites de maneira simples e prática, além de possibilitar o compartilhamento do projeto com os demais colaboradores, muito útil em trabalhos em equipe (Inclusive esse site foi feito com auxílio do figma)

HTML

Partindo para a codificação, aprendi os conceitos básicos do HTML, como as tags semânticas, além da compreensão sobre a estrutura de um documento index. Aprendi a criar páginas da web com cabeçalhos, parágrafos, listas, links e imagens, utilizando as tags apropriadas para os mecanismos de busca do navegador.

CSS

Também aprendemos o CSS, linguagem de estilização utilizada para controlar a aparência e o layout dos elementos HTML em uma página da web. Ao longo do curso, aprendemos a aplicar estilos como cores, fontes, espaçamento, bordas e posicionamento para criar páginas da web visualmente atraentes e funcionais.
