Front-End

Figma

Inicialmente, antes de aprender até mesmo linguagens de marcação, conhecemos o Figma, uma ferramenta de design colaborativa que permite que os usuários possam desenvolver a interface de seus sites de maneira simples e prática, além de possibilitar o compartilhamento do projeto com os demais colaboradores, muito útil em trabalhos em equipe (Inclusive esse site foi feito com auxílio do figma).

HTML

Partindo para a codificação, aprendi os conceitos básicos do HTML, como as tags semânticas, além da compreensão sobre a estrutura de um documento index. Aprendi a criar páginas da web com cabeçalhos, parágrafos, listas, links e imagens, utilizando as tags apropriadas para os mecanismos de busca do navegador.

CSS

Também aprendemos o CSS, linguagem de estilização utilizada para controlar a aparência e o layout dos elementos HTML em uma página da web. Ao longo do curso, aprendemos a aplicar estilos como cores, fontes, espaçamento, bordas e posicionamento para criar páginas da web visualmente atraentes e funcionais.

Back-end

Python

Partindo para a parte de Back-end, também trabalhamos com Python. A linguagem tem como característica a simplicidade e legibilidade facilitada comparada as demais concorrentes. Confesso que achei um tanto desafiador unir tal linguagem a documentos HTML, mas acredito que me sai bem de modo geral.

Arquitetura de Redes

Ao longo do curso, também tive a oportunidade de explorar a Arquitetura de Redes, um campo essencial para o funcionamento da internet e comunicação entre dispositivos. Aprendi sobre os diferentes tipos de redes, como LAN, WAN e MAN, além dos protocolos que regem a comunicação de dados e as diferentes topologias.

Exemplo de Topologia em Árvore

IoT

Também pude aprender sobre dispositivos IoT (Internet das Coisas), um campo emergente que está transformando a maneira como interagimos com a tecnologia e o mundo ao nosso redor. Aprendi sobre os diversos dispositivos conectados que compõem o ecossistema IoT, desde sensores inteligentes até aparelhos domésticos, e como eles se comunicam entre si e com a internet.
